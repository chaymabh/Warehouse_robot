# Warehouse_robot: 
 
  This code is a warehouse remote control robot code , it enables a robot to connect to WIFI ,
  get assigned speed and path from Firebase realtime database and send the data to an arduino board 
  to follow the given path with the assigned speed " vitesse " while avoiding obstacles. 
 
 
 
 ## Refrence : https://docs.arduino.cc/retired/getting-started-guides/ArduinoUnoWiFi.
