# Warehouse Robot

This repository contains code for a warehouse remote control robot. The code enables the robot to connect to WiFi, retrieve assigned speed and path information from a Firebase Realtime Database, and send this data to an Arduino board. The robot follows the given path with the assigned speed ("vitesse"), while avoiding obstacles.
